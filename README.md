# Breadboard-Friends-With-Benefits
A non-compatable sequel to Mutable Instruments' Breadboard Friends by Émilie Gillet

These devices are NOT compatable with Mutable BBF. 
These use a different breadboard power pin layout which also includes a +5v rail
The original project didn't get a lot of love and I imagined many more possible modules for the ecosystem but wanted improvements to clarity, usability and exposure of the +5v rail.

Find the original inspiration for the project here 
https://pichenettes.github.io/mutable-instruments-diy-archive/bbf/
